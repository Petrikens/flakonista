# Rusel Ruleset

This folder contains project rules and conventions used by Cursor.

- `architecture.md`: high-level architecture and module boundaries.
- `backend-api.md`: API contracts, validation, pagination, sorting, caching.
- `frontend.md`: UI conventions, composables, state management, components.
- `code-style.md`: TypeScript/Vue coding standards and review checklist.
- `performance-security.md`: performance, caching, validation, and security rules.

Tip: Keep these docs in sync with `@architecture.mdc` and server/client code.
